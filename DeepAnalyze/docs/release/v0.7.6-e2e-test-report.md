### 关键修复（本轮测试发现并修复）

1. **HS256 时间窗强制失效**（`src/domain/auth.ts:62-65`）
   ```typescript
   if (decoded.header.alg === "HS256") {
     const deadline = HUB_CONFIG.auth.hs256TransitionUntil;
     if (deadline && new Date(deadline).getTime() < Date.now()) {
       return null; // 过渡窗口已过期 — 拒绝 HS256
     }
   }
   ```
   **T_A2b** 专项回归该逻辑。

2. **SSH 部署命令注入防护**（`src/domain/worker-deployment.ts:validateDeployInputs`）
   - 对 ssh_host/ssh_user/image_tag 添加字符白名单校验
   - 拒绝 `;`, `|`, `&`, `` ` ``, `$()` 等元字符

3. **模型上传路径穿越防护**（`src/server/routes/models.ts`）
   - 拒绝包含 `..` 或绝对路径的文件名

4. **withdrawSkill 限定 submitter 范围**（`src/server/routes/marketplace.ts`）
   - UPDATE 语句增加 `AND submitter_id = $2` 防越权

- `tests/e2e/promote.spec.ts` — 简化定位器（`.first()` 现在必然指向可推广包），并改 DB 校验断言从 slug 改为 name（UI 显示 name 而非 slug）
